# Welcome to the mock data generator website!

## To deploy this project run

1. Install node.js.
```bash
brew install node
```

2. Clone git repo.
```bash
git clone git@github.com:MoojinChae/mock-polar-ai.git
```

3. Install next
```
npm install next
```

4. Run for local test
```
npm run dev
```

5. Prod Build
```bash
npm run build
```

6. Run prod build in your local.
```bash
npm start
```
